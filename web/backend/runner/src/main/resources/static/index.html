About SimpleFlow

                

                

                  

                    SimpleFlow is a minimal interpreted language built in Java with a single core data
                    structure (cell) and readable, English-like syntax. It includes a web studio for
                    fast editing and execution.

Running Help

                

                

                  
    
                    
  1. Open terminal in web/backend/runner
    2. 
                    
  2. Run ./mvnw spring-boot:run (Windows: mvnw.cmd spring-boot:run)
    3. 
                    
  3. Open http://localhost:8080/
    4. 
                    
  4. CLI: java -cp out com.simpleflow.lang.Main file.sf
    5. 
                    
  5. REPL: run java -cp out com.simpleflow.lang.Main with no args
    6.
