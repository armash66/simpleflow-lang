About SimpleFlow

                

                

                  

                    SimpleFlow is a minimal interpreted language built in Java. It focuses on readable,
                    English-like syntax with a tiny surface area for fast experimentation.
                  

                

              


              

                

                  
Commands

                

                

                  
    
                    
  • store variables
    • 
                    
  • when / otherwise conditionals
    • 
                    
  • while loops
    • 
                    
  • loop for-style loops
    • 
                    
  • next (continue) and leave (break)
    • 
                    
  • define functions + return
    • 
                    
  • print (no newline) and show (newline)
    • 
                    
  • null literal
    • 
                    
  • cell literal: @(1, 2, 3)
    • 
                    
  • length(cell) and 1-based indexing: c[1]
    • 
                  

                

              


              

                

                  
Synonyms

                

                

                  
    
                    
  • show = print with newline
    • 
                    
  • print / say = print without newline (if enabled)
